# Bio-Neural Monitor v2

Улучшённая 3D‑версия панели мониторинга с осмысленной концепцией.

## Концепция

**Синтетическая биологическая нейросеть** — система, которая обрабатывает биологические образцы через нейроподобную архитектуру. Центральное 3D‑дерево — модель нейронной структуры:

- **Корни** — точки входа данных
- **Ветви** — пути передачи (нейронные связи)
- **Узлы** — синапсы (точки обработки)
- **Кольцо орбит** — периферийные сенсоры

## Метрики (левая панель)

| Метрика | Значение |
|--------|----------|
| Активные синапсы | Количество активных нейронных соединений |
| Здоровье системы | Общий статус (0–100%) |
| Активность (ч) | Трафик данных за последние 7 часов |
| Распределение узлов | География узлов сети |

## Графики (нижний ряд)

- **Пропускная способность** — Мб/с
- **Время отклика** — латентность в мс
- **Активность узлов** — heatmap за 7 дней
- **Запросов/сек** — RPS

## 3D‑дерево

Дерево можно вращать мышью. Оно процедурно сгенерировано и визуализировано с помощью Three.js.
