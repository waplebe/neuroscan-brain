# Bio-Neural Monitor v3

Расширенная версия с живыми данными, интерактивностью и новыми возможностями.

## Новое в v3

### Живое обновление данных
- **Графики** обновляются каждые 2 секунды (пропускная способность, RPS, латентность)
- **Счётчик образцов** растёт в реальном времени (при активном мониторинге)
- **Температура и влажность** — лёгкая флуктуация
- **Системный лог** — новые записи каждые 8 секунд

### Интерактивность
- **Вращение** — перетащите мышью для поворота 3D дерева
- **Зум** — колёсико мыши для приближения/отдаления
- **Raycasting** — наведите курсор на узел или сенсор для подсказки:
  - Корневой узел
  - Синапс #1, #2, …
  - Сенсор #1, #2, …

### Управление
- **Синхронизация** — имитация процесса синхронизации, добавляет запись в лог
- **Пауза** — останавливает обновление данных и счётчика образцов
- **Экспорт** — скачивает JSON с текущим состоянием системы

### Визуальные улучшения
- **Частицы** — точки вдоль орбитального кольца
- **Повышенное дерево** — больше ветвей и узлов
- **Счётчики** — отображается количество ветвей и узлов в заголовке
- **Панель оповещений** — отдельный блок с предупреждениями
- **Вращающийся образец** — анимация 3D-сферы в правой панели
